Pagination Block Flow Sample

This sample is intentionally long so the PDF must span multiple pages. Use the checklist below to verify correct pagination behavior.

Expected PDF Behavior

  1. Blocks must appear in strict numeric order: 01 to 12.
  2. Each block is atomic and must never be split across pages.
  3. When a block does not fit at page bottom, it must start at the top area of the next page.
  4. No overlap is allowed between consecutive blocks on any page.
