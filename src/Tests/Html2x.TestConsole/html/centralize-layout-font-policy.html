Diagnostics verification

  1. Confirm a layout/margin-collapse trace appears for the sibling block cases.
  2. Confirm LayoutBuild and PdfRender lifecycle events appear in the diagnostics session.
  3. Confirm layout snapshot fragments preserve marker offsets, table row indices, and table cell column indices.
  4. Confirm the test-console envelope records the input path, output path, and raw arguments.
  5. Confirm the diagnostics JSON envelope records owner, consumer, artifact path, and approved exception evidence.
